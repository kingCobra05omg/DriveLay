Inicio de sesión

Para poder utilizar drivelay la aplicación le pedirá al usuario sus datos para poder crear una cuenta y pueda utilizar la aplicación pidiendo nombre, apellido, gmail y una contraseña segura para poder registrarse e iniciar secion.

Funcionamiento básico

Drivelay permitirá al usuario crear una lista de vehículos, también llamada "empresas", en donde el usuario podrá ingresar vehículos a la misma pidiendo patente, modelo, marca, año y un color, para poder identificar cada vehículo. Otros usuarios podrán ingresar a la empresa creada dentro de la aplicación a través de un código que el jefe les envíe, de esta forma los empleados podrán ingresar sus vehículos.

Cosas a futuro

Tenemos planeado en un futuro agregar gran variedad de nuevas funciones a la app, como por ejemplo, la implementacion de seguimiento de vehiculos, notificaciones para salida e ingreso de vehiculo , etc...
